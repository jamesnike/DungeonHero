# 主牌堆候选卡牌一览

每局游戏从以下候选池中随机抽取指定数量的卡牌组成主牌堆。

---

## 关键词说明

### 转型 (Transformation)

如果上一张**使用**（不包括弃置和回收）的牌的**类型**和该牌不同，则额外触发该卡牌的转型效果。

「使用」包含手牌打出和激活行（active row）的卡牌操作（如：激活事件、装备武器/护盾/护符、使用药水/魔法等）。

**卡牌类型分类**（用于转型判定）：

| 类型 | 说明 |
|------|------|
| instant magic | 即时魔法 |
| perm magic | 永久魔法 |
| hero magic | 英雄魔法 |
| weapon | 武器 |
| shield | 护盾 |
| amulet | 护符 |
| potion | 药水 |
| Monster Equipment | 怪物装备（劝降后装备的怪物） |
| event | 事件/建筑 |

### 侧击 (Flank)

当这张牌从手牌中打出时，如果它是手牌中**最左边**或**最右边**的牌，则额外触发该卡牌的「侧击」效果。

- 只有 1 张手牌时，该牌既是最左也是最右，**视为侧击**。
- 侧击判定在卡牌被消耗之前进行（基于打出时的手牌位置）。

### 回收袋洗回背包 (Recycle Shuffle)

将回收袋中的卡牌按瀑流规则返还背包。具体效果为：回收袋中所有卡牌的剩余瀑流次数（`_recycleWaits`）-1。只有减至 0 的牌才会移回背包，其余仍留在回收袋中继续等待。

- Perm 1 的牌若刚使用（`_recycleWaits = 0`），触发时直接返还背包。
- Perm 2 的牌刚进回收袋时 `_recycleWaits = 2`（或 1），触发后 -1，仍需继续等待。
- Perm 3 同理，需经历更多次触发才能返还。
- 背包容量不足时，已就绪的牌也会留在回收袋中。

与「立即放回」的区别：回收袋洗回背包**不是**将回收袋全部清空放入背包，而是模拟一次瀑流推进的回收效果。

### 增幅 (Amplify)

一次性魔法「增幅」。选择装备栏中的武器/护盾或手牌中的装备/伤害魔法，生成一张永久魔法（Perm 2）对该牌进行增幅：

| 目标类型 | 增幅效果 |
|----------|----------|
| 武器 | 攻击力 +1（永久修改卡面数值） |
| 护盾 | 护甲 +1（永久修改 armorMax） |
| 伤害魔法 | 伤害 +1（叠刺类增加叠刺基数，其余增加 amplifyBonus） |

- 「增幅」本身为一次性魔法，使用后进入坟场。
- 生成的「增幅：XXX」为永久魔法（Perm 2），绑定目标牌，每次使用对绑定目标增幅一次。
- 增幅修改的是卡牌本身的数值，装备被替换或卡牌被弃回后仍保留。

### Boss 形态

牌堆中的最终怪物（`isFinalMonster`）首次被击败时不会死亡，而是变身为 Boss 形态（`bossPhase`），获得以下技能：

| 技能 | 效果 |
|------|------|
| 反噬 | 每次受到伤害，对英雄造成 3 点直接伤害（无视护盾） |
| 复生 | 首次被击杀后以 1 血层复活 |
| 暴走光环 | 血层为 1 时，每个怪物回合结束，激活行所有怪物 +5 攻击并恢复 1 血层 |
| 亡灵召唤 | 被激怒时，从坟场取 4 张牌：2 怪物各占 1 个非 boss 格的顶层，2 非怪物堆叠在另一个非 boss 格上；被召唤的怪物立即激怒 |

Boss 变身时攻击力 +5，满血复活，保留原怪物的所有种族技能和强化。

## 怪物 / Monster（21 张，每局全部入选）

7 个种族，每族 3 只（共 21 只）。每个种族随机选 1 只成为精英（monsterSpecial）。

**强化等级**：每个怪物有 3 级强化（Lv.0→Lv.3），地城行怪物根据 waterfall 次数自动强化；怪物装备可通过升级系统手动升级到下一等级。
